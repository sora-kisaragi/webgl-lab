# ADR-0004: 検証用3Dモデルの取り扱い

- 日付: 2026-07-17
- ステータス: 採用

## 背景

検証に使うVRMモデル（ファンメイドモデル等）は再配布禁止のライセンスが多く、publicリポジトリにコミットできない。一方で毎回ドラッグ&ドロップするのは手間。

## 決定

- モデルは `public/models/` に置き、**`models/` を gitignore** してリポジトリに含めない
- アプリは起動時に `/models/salome.vrm` を自動読み込みする（無ければデモメッシュを表示）
- `?model=/models/foo.vrm` クエリで読み込むモデルを切り替えられる
- 各モデルの入手元とライセンスは `docs/notes/` に記録する

## 影響

- クローン直後はモデルが無い状態で動く（デモシェーダーメッシュが表示される）ため、動作確認にモデルは必須ではない
- モデルの再入手はノートに記録した入手元から各自行う
