# ADR-0003: リポジトリ運用フロー

- 日付: 2026-07-17
- ステータス: 採用

## 背景

一人での実験リポジトリであり、レビューやブランチ保護などの細かい制御は不要。速度を優先する。

## 決定

- GitHub `sora-kisaragi/webgl-lab` に **public** で公開
- **main 直コミット**（trunk-based）。PR やブランチ保護は使わない
- 設計判断は `docs/adr/` に ADR として軽量に残す
- 検証結果・調査メモは `docs/notes/` に残す（まとまってきたら GitHub Wiki への転記を検討）

## 影響

- 壊れた状態が main に乗ることは許容する。動く状態を保ちたくなったら CI を後付けする
